CODEWITHPRATIK

Code in any c++ language that can count the number of digits.

divide the number by 10 and increment ans variable by 1 till n>0

Sample code

            
  #include 
  #include 
  using namespace std;

  int main()
  {
     int n;
     cin>>n;
     int ans=0;
     while(n>0){
         ans++;
         n=n/10;
     }
  }

Sample output

    Input n varaible from user
    create while loop for n>0 :
    divide n by 10
    increment ans variable

Variables used in program-
In the above code total 2 variables used.
n stores the number whose digits are to be found
Simillarly ans stores the count of digits in the number

Time Complexity-

Since there is one loop so time complexity of above code is O(no of digits of n)
